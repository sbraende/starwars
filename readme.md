## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/star-wars-explorer.git
```

2. Navigate into the source directory:

```bash
cd star-wars-explorer
```

3. Install dependencies

```bash
npm install
```

4. Run local server:

```bash
npm run dev
```
